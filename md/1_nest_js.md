## nest.js?
효율적이고 확장 가능한 Node.js 서버 측 애플리케이션을 구축하기 위한 프레임워크이다. 

### nest.js 설치하기

1. Node.js 설치
2. Nest JS CLI 로 시작하기
    * nest.js 클라이언트를 설치해야한다.
    ```shell
    npm i -g @nestjs/cli
    nest new project-name
    ```

### nest.js 실행하기

`package.json` 파일에 가면, 설정되어 있는 많은 명령어들을 볼 수 있다. `npm run` 과 함께 뒤에 붙여주자. 그러면 실행된다.

```
npm run start:dev
```
